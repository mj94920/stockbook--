## 1. 제품 개요

### 제품명
StockBook — 나만의 주식 포트폴리오 관리 앱

### 한 줄 설명
국내·해외 주식 포트폴리오를 하나의 화면에서 관리하고, 매매 기록·투자일지·복리 목표 달성 여부를 추적하는 개인용 앱

### 플랫폼
| 플랫폼 | 방식 | 업데이트 |
|---|---|---|
| 데스크톱 (Windows) | Electron 포터블 exe | `npm run build` 후 exe 교체 |
| 모바일 (Android) | TWA + GitHub Pages PWA | `git push`만으로 자동 갱신 |

### 데이터 저장
- **데스크톱**: `%APPDATA%\StockBook\stockbook-data.json` (JSON 단일 파일)
- **모바일**: 브라우저 localStorage (서비스워커 network-first 전략)

---

## 2. 핵심 사용자 및 투자 원칙

- **사용자**: 개인 투자자 (국내 주식 중심, 해외 일부 병행)
- **매매 스타일**: 5일선·10일선 기반 단기 타이밍 + 20일·200일선 중장기 추세 필터
- **목표 수익률**: 월 4.5% 복리 (월별 달성 여부 추적)

## 3. 주요 기능 명세

### 3-1. 포트폴리오 탭

#### 포트폴리오 테이블
- 보유 종목 목록: 종목명, 평균단가, 수량, 현재가, 평가금액, 수익률, 섹터, 포지션
- 현재가 셀 하단 sub-text: **평균단가 대비** 손익 표시
  - 이익: 빨간색 `+N원 (+X.XX%)`
  - 손실: 파란색 `-N원 (-X.XX%)`
- 매도 버튼, 종목 편집, 목표가·손절가 설정

#### 시세 자동 조회
- **국내 종목(6자리 코드)**: 네이버 금융 모바일 API 우선 → 실패 시 Yahoo Finance `.KS`→`.KQ` 폴백
- **해외 종목**: Yahoo Finance v8 Chart API 직접 호출
- 저장 데이터: `currentPrice`, `prevClose`(전일종가), `openPrice`(당일 시가)
- 🔄 시세 새로고침 버튼 (전체 일괄), 개별 모달 내 자동 조회 버튼

#### 매수 등록 폼
- 종목명 선택 시 섹터·포지션 자동완성 + 현재가 자동 조회
- 티커 미등록 종목은 현재가 수동 입력 유지

#### 포트폴리오 바 차트
- 섹터별 비중 시각화

---

### 3-2. 잔고·예수금 탭

#### 총자산 관리
- 총자산(totalAsset) 수동 입력
- 포트폴리오 평가금액 + 예수금 자동 합산 표시

#### 예수금 탭
- 입금/출금 내역 등록 폼
- 잔액 자동 계산: 입금 − 출금 − 현금매수 + 현금매도

---

### 3-3. 거래기록 탭

- 매수/매도 등록 시 자동 생성 (수동 등록 폼 제거)
- 종목별 필터, 날짜순 정렬
- 🧹 고아 데이터 정리 버튼: 포트폴리오에 없는 종목의 잔존 기록 일괄 삭제

---

### 3-4. 투자일지 탭

- 매수 시 buy 타입 일지 자동 생성
- 매도 시 기존 buy 일지에 매도일·투자기간·수익률 자동 기입, 없으면 sell 타입 초안 생성
- 멘털 상태 기록 (원칙 지켰어 / 흔들렸네 / 고점에 팔았다 등)

---

### 3-5. 월복리 시뮬레이터 탭

#### 기본 계산
- 입력: 원금, 목표 월 수익률(%), 기간(개월), 월 추가납입, 시작일
- 출력: 월별 목표잔액, 최종 자산, 총 투입금, 수익, 누적수익률, 연환산수익률
- **입력값 영구 저장**: 앱 재시작 후 자동 복원·재계산 (`state.mcSettings`)

#### 월별 실적 추적 (시작일 입력 시 활성화)
| 열 | 내용 |
|---|---|
| 월 | 달력 기준 연월 (현재 달: 파란 배경 + ◀현재) |
| 목표잔액 | 복리 계산 목표값 |
| 실제잔액 | 스냅샷 or 현재 총자산 (클릭하여 수동 입력) |
| 실제수익률 | 전월 대비 실제 수익률 (초록/노랑/빨강) |
| 달성 | ✅ (실제 ≥ 목표) / ❌ (미달) / — (미입력) |

- **📌 이번 달 실적 기록** 버튼: 현재 총자산을 이번 달 스냅샷으로 저장
- **실제잔액 셀 클릭** → 전용 모달로 과거 달 금액 수동 입력 (0 입력 시 삭제)
- 시작일 없으면 기존 수동 체크(✅/—) 방식 유지

---

### 3-6. 물타기 계산기 탭

- 현재 보유 수량·단가 입력 → 추가 매수 후 평균단가 자동 계산
- 모바일 서브탭 스크롤 지원 (7개 탭 가로 스크롤)

---

### 3-7. 뉴스·공시 탭

#### 종목 검색
- 검색창에 종목명/코드 입력 → DART, 네이버 뉴스, Investing.com 직접 검색

#### 링크 그리드
- DART 전자공시 (사업보고서, 분기보고서, 주요사항보고서)
- 네이버 금융 (시황, 뉴스 통합)
- Investing.com (해외 종목, 거시경제)

#### 보유 종목별 바로가기 (하단 카드)
- 국내 종목: 📋DART · 💹네이버 금융
- 해외 종목: 📋DART · Inv.
- (제거된 항목: 📰뉴스 별도 버튼, KRX KIND 버튼)

## 6. 미결 과제 및 향후 계획

### 기술 부채
- Yahoo Finance CORS 정책 변경 시 Cloudflare Worker 프록시로 전환 필요
- 네이버 금융 API CORS 실환경 검증 (현재 동작 확인, 정책 변경 가능성 있음)

### 향후 고려 기능
- 데이터 저장 경로를 exe 위치 기준으로 변경 (완전 이식성 확보, Option B)
- 복리 시뮬레이터 "초기 자산 체크포인트" 기능 — 시작 시점 실제 잔액 설정
- 종목별 차트 연동 (네이버 금융 차트 iframe 또는 딥링크)
- 백업/복원 기능 (JSON 내보내기·가져오기)
